Question: Discuss the scope of var, let, and const

    Var : Var means variable. var variable is sometimes a global scope & sometimes a local scope. When it's declared in a function then it is a local scope. That means that variable can only be used in the function.But if the var is declared outside of a function than it's called gloabal variable means that var can be called from anywhere.

    Let: Let means variable.let variable is a block scope. Let variable is only can be used in the function & it's can't be redeclared in the same block.

     

    const : Const is also block scope as let variable.But there is an extra feature. which is when const variable is declared it cannot be redeclared in the whole code.
